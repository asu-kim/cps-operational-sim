# Drone Simulation Results

## Overview

This directory is reserved for simulation-specific output notes. In the current repository layout, shared drone result PDFs are stored under:

```text
results/drone/simulation/
```

and RC command outputs are stored under:

```text
data/drone/rc-out/
```

## Typical Outputs

- `rc-out.csv`, `rc-out-2.csv`, `rc-out-3.csv`, `rc-out-4.csv`: RC command logs generated by LF simulation.
- `*-path-obstacles.pdf`: single-run drone path and obstacle plot.
- `*_overlay.pdf`: two-run comparison plot.

## Regenerate Plots

From the repository root:

```bash
python3 tools/plot_drone_path_compare.py data/drone/rc-out/rc-out.csv data/drone/raw-logs/Data1 data/drone/rc-out/rc-out-2.csv data/drone/raw-logs/Data2 results/drone/simulation/original_vs_obstacle_injected_overlay.pdf
```

## Notes

- Pair each RC output with the ToF dataset that generated it.
- Each ToF directory must contain `front.csv`, `left.csv`, `right.csv`, `top.csv`, and `bottom.csv`.
- The plotted path is reconstructed from RC commands and should be used for relative comparison.
